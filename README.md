# Crop Recommendation Using Random Forest

This project demonstrates how the Random Forest machine learning algorithm can be used to recommend suitable crops based on soil nutrients and climate conditions such as nitrogen, phosphorus, potassium, temperature, humidity, rainfall, and soil pH.

The tutorial explains how the model was trained, evaluated, and interpreted using feature importance and permutation importance techniques.

## Project Contents

- Google Colab notebook containing the implementation
- Dataset used for training the model
- Tutorial report explaining the workflow and results

## How to Run the Project

1. Open the notebook using Google Colab
2. Run all cells step-by-step
3. The output will reproduce the results shown in the tutorial
